## 🔧 Command Reference

### Basic:
```bash
python backlink-generator.py DOMAIN
```

### With options:
```bash
python backlink-generator.py DOMAIN [OPTIONS]

Options:
  --test                 Test if URLs are accessible
  --output FILE          Save to specific CSV file
  --limit N              Show only N results
  --shuffle / --no-shuffle  Shuffle templates
  --config-dir DIR       Template directory
```

### Advanced:
```bash
python advanced-backlink-generator.py DOMAIN [OPTIONS]

Options:
  --batch FILE           Process multiple domains from file
  --analyze              Analyze template performance
  --test                 Test URLs
  --workers N            Concurrent workers (default: 10)
  --report               Generate detailed report
  --export-formats FMT   Export formats (csv,json,txt)
  --output-dir DIR       Output directory
```
